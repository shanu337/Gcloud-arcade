# 🌐 VPC (Virtual Private Cloud)

## 🧠 What is VPC?
A VPC is a private network inside Google Cloud where you can run your resources securely.

---

## 🔑 Key Concepts

### 🧩 Subnets
- Divide VPC into smaller networks
- Regional (exist in specific regions)

### 🌍 IP Addressing
- Internal IP → communication within VPC
- External IP → internet access

### 🔗 VPC Peering
- Connect two VPC networks
- Allows private communication without internet
